# Cat Steal Fish 🐱🐟

## 프로젝트 소개

`cs1graphics` 라이브러리를 이용하여 제작한 간단한 애니메이션 프로젝트입니다.

고양이가 생선가게에서 생선을 훔쳐 달아나는 장면을 도형(`Circle`, `Rectangle`, `Polygon`, `Ellipse`)과 `Layer`를 활용하여 구현하였습니다. 객체를 레이어로 관리하고, 이동 애니메이션을 통해 장면이 자연스럽게 이어지도록 구성하였습니다.

## 주요 기능

* 고양이, 생선, 사람, 생선가게를 도형으로 직접 구현
* `Layer`를 활용한 객체 관리
* `move()`와 `moveTo()`를 이용한 애니메이션 구현
* `sleep()`을 이용한 시간 지연 효과
* 객체의 `Depth`를 조절하여 화면의 앞뒤 관계 표현

## 개발 환경

* Python
* cs1graphics

## 실행 방법

```bash
python cat.py
```

> **참고**
>
> 본 프로젝트는 `cs1graphics` 라이브러리를 사용합니다. 실행을 위해서는 해당 라이브러리가 포함된 환경이 필요합니다.
